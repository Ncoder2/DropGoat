DropGoat

DropGoat is an intentionally vulnerable web application using DropWizard framework for Java.

GitHub

Vulnerable endpoints

  • XSS in unprotected HTML tag content
  • XSS in unprotected HTML attribute
  • XSS in unprotected JavaScript code
  • XXE in XML parser

Protected endpoints

  • Non-exploitable XSS in HTML tag content escaped by owasp.Encode
  • Non-exploitable XSS in HTML attribute escaped by owasp.Encode
  • Non-exploitable XSS in HTML tag content auto-escaped by Freemarker template
